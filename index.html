TurtleJS app

Commands

Environment functions

newTurtle(name)

  • Description: Creates a new turtle and associates a name with it. The new turtle will appear on the center of the drawing space.
  • Important note 1: The environment already provides a default turtle called "t".
  • Important note 2: The multi-turtle subsystem is still an experimental feature of TurtleJS.
  • Example: newTurtle("greenTurtle");

wait(function, interval)

  • Description: Waits for the time specified on the interval parameter (of type Number) and calls the function passed as the first parameter. It's equivalent to JavaScript's setTimeout function.
  • Example: wait(function(){t.lt(90); t.fw(100);}, 500);

Space's functions

clear()

  • Description: Clears all the drawings in the space, but does not change the turtles' positions and appearances.
  • Example: clear();

bkColor(red, green, blue)

  • Description: Changes the space's background color. The red, green and blue parameters are all of type Number. They range from 0 to 100, each one specifying the percentage for each color component.
  • Example: bkColor(100, 0, 0); // Sets the background color to red.

maxX()

  • Description: Returns the maximum horizontal (x) coordinate of the 2D space.
  • Example: t.go(maxX(), maxY());

maxY()

  • Description: Returns the maximum vertical (y) coordinate of the 2D space.
  • Example: t.go(maxX(), maxY());

Turtle's positioning methods and properties

forward(distance) / fw(distance)

  • Description: Moves the turtle forward with the specified distance. Negative distance values will move the turtle backwards.
  • Example: t.fw(100);

back(distance) / bk(distance)

  • Description: Moves the turtle backwards with the specified distance. Negative distance values will move the turtle forward.
  • Example: t.bk(100);

left(angle) / lt(angle)

  • Description: Rotates the turtle counterclockwise with the specified angle. This rotation is relative to the current turtle's heading. Negative angle values will rotate the turtle clockwise.
  • Example: t.lt(90); //Rotates 90 degrees.

right(angle) / rt(angle)

  • Description: Rotates the turtle clockwise with the specified angle. This rotation is relative to the current turtle's heading. Negative angle values will rotate the turtle counterclockwise.
  • Example: t.rt(90); //Rotates 90 degrees.

go(x, y)

  • Description: Absolute positioning. Moves the turtle to the specified coordinates.
  • Note: The upper left corner of the drawing space is (0, 0). The vertical (y) coordinates increases from top to bottom.
  • Example: t.go(maxX()/3, maxY() - 200);

home()

  • Description: Moves the turtle to the center of the drawing space.
  • Example: t.home();

x [Number, read-only property]

  • Description: Returns the turtle's current horizontal (x) coordinate.
  • Example: t.go(t.x, 200); //Moves the turtle only on the vertical axis.

y [Number, read-only property]

  • Description: Returns the turtle's current vertical (y) coordinate.
  • Example: t.go(200, t.y); //Moves the turtle only on the horizontal axis.

heading [Number, property]

  • Description: Allows to set (and get) the absolute heading angle of the turtle.
  • Example: t.heading = t.heading + 45; //Same as t.rt(45)

Turtle's appearance methods and properties

visible [Boolean, property]

  • Description: Sets/gets the visibility of the turtle itself, so it can be shown or hidden.
  • Example: t.visible = !t.visible; //Toggles turtle's visibility.

scale [Number, property]

  • Description: Allows to set (and get) the scale of the turtle's image. It's default value is 1.
  • Example: t.scale = t.scale*3.5;

image [String, property]

  • Description: Sets/gets the image of the turtle. Provided images on this (very first) version are "turtle" (default), "rocket" and "turtleSolid".
  • Example: t.image = "rocket";

show()

  • Description: Convenience method equivalent to setting visible to true.
  • Example: t.show();

hide()

  • Description: Convenience method equivalent to setting visible to false.
  • Example: t.hide();

Turtle's drawing methods and properties

clear() / cl()

  • Description: Clears all the turtle's drawings.
  • Example: t.cl();

penWidth / pw

  • Description: Sets/gets the pen's width.
  • Example: t.pw = 5;

pc(red, green, blue)

  • Description: Sets/gets the pen's color. The red, green and blue parameters are all of type Number. They range from 0 to 100, each one specifying the percentage for each color component.
  • Example: t.pc(100, 50, 100);

isPenDown: Bool

  • Description: Returns true if the turtle's pen is currently down.
  • Example: t.visible = t.isPenDown;

penUp() / pu()

  • Description: Sets the turtle's pen up.
  • Example: t.pu();

penDown() / pd()

  • Description: Sets the turtle's pen down.
  • Example: t.pd();

Turtle's combined drawing and position methods

reset()

  • Description: This is a convenience method that calls clear() and home(), in that order: it clears all the turtle's drawings and sends the turtle to home (the center of the drawing space).
  • Example: t.reset();
